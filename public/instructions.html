⚠️ This app is only compatible with PCSX2.
⚠️ Buzz!: The Music Quiz (the first game in the franchise) does not support 8 players — it only supports 4.

1. Players join

  1. Open your phone camera and scan the QR code, or type the address in your browser.
  2. Pick a free slot (Player 1–8).
  3. Enter your name when asked.
  4. The controller screen appears — you're ready to play!

2. Configure PCSX2 — 4 players (USB 1)

  1. Open PCSX2 and go to Settings → Controllers.
  2. Click USB Port 1.
  3. In the dropdown, choose Buzz! Controller.
  4. Map each player's buttons (1–4) using the table below.

3. For 8 players — add USB 2

  1. Still in Settings → Controllers, click USB Port 2.
  2. Choose Buzz! Controller again in the dropdown.
  3. Map players 5–8 using the keys in the table.

4. Play

  1. Start the game in PCSX2.
  2. Keep the PCSX2 window in focus — required to receive button presses.
  3. Press buttons on your phone and have fun!

⚠️ Esta app é apenas compatível com o PCSX2.
⚠️ Buzz!: The Music Quiz (o primeiro jogo da franquia) não é compatível com 8 jogadores — suporta apenas 4.

1. Os jogadores entram

  1. Aponta a câmara ao QR code ou abre o endereço no browser.
  2. Escolhe um lugar livre (Jogador 1–8).
  3. Escreve o teu nome quando solicitado.
  4. O ecrã do comando aparece — já está pronto a usar!

2. Configurar o PCSX2 — 4 jogadores (USB 1)

  1. Abre o PCSX2 e vai a Settings → Controllers.
  2. Clica em USB Port 1.
  3. No menu dropdown escolhe Buzz! Controller.
  4. Mapeia os botões de cada jogador (1–4) conforme a tabela abaixo.

3. Para 8 jogadores — adiciona USB 2

  1. Ainda em Settings → Controllers, clica em USB Port 2.
  2. Escolhe novamente Buzz! Controller no dropdown.
  3. Mapeia os jogadores 5–8 com as teclas da tabela.

4. Jogar

  1. Inicia o jogo no PCSX2.
  2. Mantém a janela do PCSX2 em foco — é necessário para receber os botões.
  3. Carrega nos botões no telemóvel e divirte-te!
